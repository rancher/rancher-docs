Rancher 通过各种卷插件支持持久化存储。不过，在使用这些插件将持久存储绑定到工作负载之前，您必须先配置存储本身，无论是服务提供商提供的基于云的解决方案，还是您自己管理的内部部署解决方案。

Rancher 提供了有关如何配置一些常用存储方法的文档：

- [NFS](nfs-storage.md)
- [vSphere](vsphere-storage.md)
- [EBS](persistent-storage-in-amazon-ebs.md)
